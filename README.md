# Algorithms in Go

Practising Go programs from examples in the book "Graphic Go Algorithms" by Yang Hu

1. Added testing
2. Using go modules
3. Learn how to structure go code

To build:
```
// output executable to the bin folder
go build -o bin ./
```

To run:
```
./bin/graphicgo
```

To test:
```
go test
```

To clean:
```
git clean -dxf
```
